# Flatten Pitch Curve

## 功能

将选中音符或选中音符组范围内的音高曲线抹平。脚本会把 `pitchDelta` 参数曲线写回 0 cents，并可选移除范围内的 Synthesizer V Studio 2 音高控制点/曲线。

会处理：

- 选中音符：只处理当前音符组中这些音符覆盖的时间范围。
- 选中音符组：处理该音符组引用实际覆盖的目标时间范围。
- `pitchDelta` 参数：删除范围内旧点，并在范围边界写入 0 cents。
- Studio 2 音高控制对象：移除与范围重叠的 Pitch Control Point / Curve。

## 用法

1. 在钢琴窗中选中需要抹平音高曲线的音符，或在轨道中选中音符组。
2. 运行 `Flatten Pitch Curve`。
3. 选择处理范围：选中音符、选中音符组，或两者一起处理。
4. 选择是否抹平 `pitchDelta` 曲线。
5. 选择是否移除范围内 Studio 2 音高控制点/曲线。
6. 点击 OK 执行。

## 注意事项

- 脚本不会改变音符音高、歌词、音符长度或音符位置。
- `pitchDelta` 会被压回 0 cents；自然换音、声库自动生成的音高过渡和颤音不属于 `pitchDelta` 参数点，可能仍会存在。
- 默认会在选区外相邻位置写入保护点，减少选区内抹平对外侧曲线的影响。
- 处理音符组时，脚本修改的是音符组目标。如果该目标被多个引用复用，其他引用也会同步变化。
- Studio 2 音高控制 API 只在支持该功能的版本中可用；不支持时脚本会跳过并在完成提示中说明。
