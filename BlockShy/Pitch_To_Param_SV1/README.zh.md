# Pitch to Parameter (SV1)

## 功能

将选中音符的音高信息映射到目标参数曲线，用于把旋律音高或弯音转换为张力、气声、性别、发声、颤音包络、响度等表现参数。

弹窗会显示内置候选参数，并在运行时按当前音符组校验目标参数是否可用。内置候选包括：

- Tension / 张力
- Breathiness / 气声
- Gender / 性别
- Voicing / 发声
- Vibrato Envelope / 颤音包络
- Loudness / 响度
- Tone Shift / 声区转换

也可以填写自定义参数名，用于尝试访问当前版本或声库支持的其他参数。填写 `Cool`、`Dark` 这类唱法名称时，脚本会自动尝试对应的 `vocalMode_Cool`、`vocalMode_Dark` 自动化参数；也可以直接填写完整的 `vocalMode_名称`。

## 用法

1. 在钢琴窗中选中需要处理的音符。
2. 从脚本菜单运行 `Pitch to Parameter (SV1)`，在弹窗中填写参数。
3. 选择目标参数，或填写自定义参数名。
4. 选择音高来源：
   - “轻量：音符音高 + pitchDelta”：速度快，包含音符音高、音符 detune、音符组 pitch offset 和 `pitchDelta`。
   - “仅跟随 pitchDelta”：忽略旋律音高，只把弯音映射到目标参数。
5. 选择点密度：
   - “智能精简”：按采样间隔采样，再用线性误差精简冗余点。
   - “保留全部采样点”：写入所有采样点。
   - “强制线性”：每个音符只写入首尾两个点。
6. 选择写入模式：
   - “覆盖选中音符范围”：默认模式，先清理选中音符范围内的旧目标参数点。
   - “仅追加/更新同位置点”：不清理旧点，只写入新点。
   - “清空目标参数后重建”：删除目标参数的所有旧点，再写入本次生成点。
7. 设置采样间隔、精简阈值、中心音高、映射强度和方向。映射强度的单位是“每半音变化对应多少目标参数单位”，因此声区转换或唱法参数这类大范围参数通常需要比张力、气声更大的强度值。
8. 点击 `OK` 执行。

## 注意事项

- 参数值会按目标参数官方定义的范围自动限制，不再使用固定的 `-1.0` 到 `1.0` 范围。
- 本目录是 SV1/旧版本专用脚本，静态 metadata 只暴露传统 `main()` 弹窗入口，不声明侧边栏类型。
- 如需 SV2.1.2+ 侧边栏版本和“计算后音高 (Studio 2)”来源，请使用 `BlockShy/Pitch_To_Param_SV2/Pitch_To_Param_SV2.lua`。
- SV1 版本只提供“轻量：音符音高 + pitchDelta”和“仅跟随 pitchDelta”。
- 默认写入模式会清理选中音符范围内的旧目标参数点，重复运行结果更可控。
- 参数写入当前音符组目标。如果该目标被多个引用复用，其他引用也会同步变化。
- 脚本会校验 `Automation:getType()` / `Automation:getDefinition()` 返回的真实参数类型，避免不可用的自定义名称意外写入 `pitchDelta`。
- `Tone Shift` / 声区转换优先作为自动化曲线写入；如果当前宿主不提供对应自动化曲线，但支持 `NoteGroupReference:setVoice()` 的 `paramToneShift`，脚本会改为把生成值的平均值写入当前音符组引用的声线属性。这个回退路径不是时间曲线，写入模式和选中范围清理不适用。
- `vocalMode_名称` 和其他自定义参数名的可用性取决于当前 Synthesizer V Studio 版本和声库支持。
