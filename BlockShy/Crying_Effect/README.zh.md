# Crying Effect

## 功能

为选中音符快速添加哭腔风格的表现参数。脚本会组合调整颤音包络、气声、张力，并可选添加尾部音高下坠。

会写入：

- `vibratoEnv`：增强颤音深度
- `breathiness`：增加气声
- `tension`：增加张力并加入少量随机抖动
- `pitchDelta`：可选尾部下坠

## 用法

1. 在钢琴窗中选中一个或多个音符。
2. 运行 `Crying Effect`。
3. 调整哭腔强度。
4. 按需勾选“添加尾部下坠”。
5. 点击 OK 执行。

## 注意事项

- 脚本会直接写入参数点，不会自动清理选区内已有参数。
- 张力参数包含随机抖动，因此每次运行的结果可能略有不同。
- 极短音符上生成的效果可能比较密集，必要时可手动整理参数点。
