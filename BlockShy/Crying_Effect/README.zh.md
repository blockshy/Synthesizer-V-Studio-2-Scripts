# Crying Effect

## 功能

为选中音符生成哭腔风格的表现参数。新版脚本提供可直接使用的哭腔预设，并会按预设写入颤音、气声、张力和音高哭腔曲线。

可写入：

- `vibratoEnv`：在音符中后段增强颤音包络，并在首尾恢复到标准值。
- `breathiness`：按起势、峰值、回落位置生成气声包络。
- `tension`：按包络生成张力，并可加入可控随机量。
- `pitchDelta`：可选音高哭腔，包括轻微上扬、下探、尾部摆动和相对当前曲线的尾部下坠。

## 预设

- “轻微哽咽”：较克制，适合不想破坏原唱法的局部修饰。
- “自然哭腔（推荐）”：默认预设，兼顾颤音、气声、张力和少量音高波动。
- “明显哭腔”：哭腔特征更清楚，适合副歌或情绪句。
- “强烈哭腔”：更大的音高起伏、张力和气声，适合夸张表达。
- “尾音哽咽”：偏向尾部处理，默认只让每段选区的最后音符产生明显尾坠。
- “自定义”：使用下方高级包络和下坠参数。

## 用法

1. 在钢琴窗中选中一个或多个音符。
2. 打开 Scripts 侧边栏中的 `Crying Effect` 面板。
3. 选择哭腔预设。默认“自然哭腔（推荐）”可以直接使用。
4. 勾选需要生成的模块：颤音包络、气声、张力、尾部下坠。
5. 设置预设强度倍率和写入模式。
6. 如果选择“自定义”，调整包络位置：
   - 起势位置
   - 峰值位置
   - 回落位置
7. 按需设置张力随机量、是否固定随机结果、下坠开始位置、下坠深度、是否只处理每段选区最后一个音符。
8. 可点击 `Refresh` 更新选中音符数量。
9. 点击 `Run` 执行。

## 写入模式

- “覆盖选中音符范围”：默认模式。会清理选中音符范围内已启用参数的旧点，再写入新点。
- “仅追加/更新同位置点”：不清理旧点，只写入本次生成点。
- “清空已启用参数后重建”：删除已启用参数的所有旧点，再写入本次生成点。

## 注意事项

- 参数值会按官方 automation 范围自动限制，避免写入超出范围的值。
- 本脚本是 Synthesizer V Studio 2.1.2+ 侧边栏脚本，不再通过顶部 Scripts 菜单弹窗运行。
- 固定随机结果默认开启，重复运行时张力随机形状更可复现。
- 预设会覆盖高级包络和下坠参数；高级滑块主要用于“自定义”预设。
- 音高哭腔基于当前 `pitchDelta` 叠加，不再把曲线强制拉回 0。
- 尾部下坠深度会乘以预设强度倍率和预设内置强度。
- 参数写入当前音符组目标。如果该目标被多个引用复用，其他引用也会同步变化。
- “尾后恢复音高偏移”会在音符结束后额外写一个恢复点，可能影响紧接在后的音符，请按场景使用。
