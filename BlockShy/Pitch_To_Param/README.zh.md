# Pitch to Parameter

## 功能

将选中音符的音高变化映射到目标参数曲线，用于把旋律高低或弯音转换为张力、气声、性别、音色或响度等表现参数。

支持目标参数：

- Tension / 张力
- Breathiness / 气声
- Gender / 性别
- Tone Shift / 音色
- Loudness / 响度

## 用法

1. 在钢琴窗中选中需要处理的音符。
2. 运行 `Pitch to Parameter`。
3. 选择目标参数。
4. 选择映射逻辑：
   - “全音高跟随”：旋律音高和弯音都会影响参数。
   - “仅跟随弯音”：忽略旋律音高，只跟随 `pitchDelta`。
5. 选择点密度：
   - “智能精简”：保留变化细节，同时减少冗余点。
   - “强制线性”：每个音符只写入首尾两个点。
6. 设置中心音高、映射强度和方向后点击 OK。

## 注意事项

- 脚本会直接向目标参数曲线写入点，不会自动清理选区内已有参数点。
- 响度以外的参数会限制在 `-1.0` 到 `1.0` 范围内。
- `Tone Shift` 的可用性取决于当前 Synthesizer V Studio 版本和声库支持。
