# Pitch to Parameter

## 功能

将选中音符的音高信息映射到目标参数曲线，用于把旋律音高、弯音或 Synthesizer V Studio 2 计算后的音高转换为张力、气声、性别、清浊、颤音包络、响度等表现参数。

侧边栏面板会显示内置候选参数，并在运行时按当前音符组校验目标参数是否可用。内置候选包括：

- Tension / 张力
- Breathiness / 气声
- Gender / 性别
- Voicing / 清浊
- Vibrato Envelope / 颤音包络
- Loudness / 响度
- Tone Shift / 音色，作为兼容性尝试

也可以填写自定义参数名，用于尝试访问当前版本或声库支持的其他参数。

## 用法

1. 在钢琴窗中选中需要处理的音符。
2. 打开 Scripts 侧边栏中的 `Pitch to Parameter` 面板。
3. 选择目标参数，或填写自定义参数名。
4. 选择音高来源：
   - “轻量：音符音高 + pitchDelta”：速度快，包含音符音高、音符 detune、音符组 pitch offset 和 `pitchDelta`。
   - “仅跟随 pitchDelta”：忽略旋律音高，只把弯音映射到目标参数。
   - “计算后音高”：使用 Synthesizer V Studio 2 的计算后音高，能更接近实际演唱音高；如果音高尚未计算完成，脚本会提示或回退。
5. 选择点密度：
   - “智能精简”：按采样间隔采样，再用线性误差精简冗余点。
   - “保留全部采样点”：写入所有采样点。
   - “强制线性”：每个音符只写入首尾两个点。
6. 选择写入模式：
   - “覆盖选中音符范围”：默认模式，先清理选中音符范围内的旧目标参数点。
   - “仅追加/更新同位置点”：不清理旧点，只写入新点。
   - “清空目标参数后重建”：删除目标参数的所有旧点，再写入本次生成点。
7. 设置采样间隔、精简阈值、中心音高、映射强度和方向。
8. 可点击 `Refresh` 用当前选中音符更新建议中心音高。
9. 点击 `Run` 执行。

## 注意事项

- 参数值会按目标参数官方定义的范围自动限制，不再使用固定的 `-1.0` 到 `1.0` 范围。
- 本脚本是 Synthesizer V Studio 2.1.2+ 侧边栏脚本，不再通过顶部 Scripts 菜单弹窗运行。
- 面板标题和界面文本可通过“语言 / Language”切换中文或英文，默认中文；宿主侧边栏脚本列表名称来自静态 metadata，不能随面板开关实时切换。
- 面板默认显示完整控制项；“功能与用法”说明默认隐藏，可通过“显示功能与用法”展开。
- 默认写入模式会清理选中音符范围内的旧目标参数点，重复运行结果更可控。
- “计算后音高”依赖 Synthesizer V Studio 的音高计算状态；如果采样缺失，完成提示会显示回退数量。
- 参数写入当前音符组目标。如果该目标被多个引用复用，其他引用也会同步变化。
- `Tone Shift` 和自定义参数名的可用性取决于当前 Synthesizer V Studio 版本和声库支持。
